4. Jelaskan bagaimana cara kerja mekanisme pembelajaran mesin dalam model pembelajaran mesin yang menggunakan teknik deep learning dengan menggunakan metode backpropagation dan menghitung gradient descent pada setiap lapisan jaringan neuron dengan menggunakan notasi vektor dan matriks serta menjelaskan perbedaan antara proses training dan inference dalam model tersebut?


Jawaban:

Mekanisme pembelajaran mesin dalam model pembelajaran mesin menggunakan teknik deep learning adalah dengan menggunakan metode backpropagation. Backpropagation adalah suatu metode untuk menghitung gradient descent pada setiap lapisan jaringan neuron dengan menggunakan notasi vektor dan matriks. Proses pelatihan terdiri dari forward propagation dan backward propagation, dimana kedua proses ini digunakan untuk mengupdate parameter dari model dengan cara mengesktrak informasi dari data pelatihan. Sedangkan inference berfokus pada penggunaan model untuk membuat prediksi atas data baru. Proses inference hanya menggunakan forward propagation saja dan tidak mengupdate parameter.


Penjelasan:

Backpropagation adalah suatu metode yang menggunakan notasi vektor dan matriks yang digunakan untuk menghitung gradient descent pada setiap lapisan jaringan neuron. Proses training melibatkan forward propagation dan backward propagation untuk mengupate parameter dari model. Sementara itu, proses inference hanya menggunakan forward propagation saja tanpa mengupdate parameter.
