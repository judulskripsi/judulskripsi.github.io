14. Ketika anda sudah mengunggah file website, tetapi di saat anda mencoba mengakses website masih belum tampil, apa saja yang menyebabkan terjadinya hal tersebut?


Beberapa faktor yang bisa menyebabkan website tidak bisa tampil dengan benar :

File tidak terupload ke direktori publik_html. Semua file website dan folder harus berada dibawah direktori publik_html (atau lengkapnya/home/username/publik_html). Di luar direktori tersebut file website tidak akan dapat ditampilkan. Pastikan anda telah masuk ke direktori publik_html dahulu baru mengupload file.Salah tempat upload. Jika mengunakan FTP, pertama kali setelah software FTP anda terkoneksi dengan server web hosting, direktorinya adalah /home/username/. Anda perlu mengklik folder publik_html dahulu agar bisa masuk ke /home/username/publik_html.Tidak ada file index. Nama file yang pertama kali akan ditampilkan sebagai halaman depan haruslah index. *(index.htm, index.html, atau index.php). Tanpa file index, halaman depan (homepage) website anda tidak akan tampil.Penulisan nama index file salah. Masih berkaitan dengan poin 2 diatas, bisa jadi anda telah memiliki file index. Akan tetapi penamaanya salah (mengandung huruf kapital). Nama file index harus dalam huruf kecil. Contoh nama file yang benar : index.html / index.html bukan Index.html / index.HTML.Framework. Terkadang beberapa framework mempunyai cara sendiri untuk setup website pada hosting. Silahkan mencari informasinya lebih lanjut, jika kamu memakai framework dalam pembuatan websitenya.lainnya mungkin, database yg tidak terkoneksi dgn benar jika memakai db, ada kesalahan sintaks pada penulisan program, dan lain-lain.

23. jelaskan, Mengapa disebutkan bahwa HTML hanya untuk isi sedangkan CSS hanya untuk tampilan pada sebuah website?​


Jawaban & Penjelasan:

Dalam dunia pemrograman web, kita mengistilahkan HTML sebagai organ badan suatu halaman website dan CSS sebagai kulit yang mempercantik serta JS sebagai rangka badan yang menggerakkan dan memberi fungsi interaktif kepada website tersebut. HTML sudah menjadi balik layar dari website yang kita kunjungi, coba saja view-source. HTML-lah yang digunakan untuk membuat tatanan sebuah halaman web. Oleh karena itu, HTML dikatakan 'hanya untuk isi' yang berarti menjadi dasar suatu halaman website. Sedangkan CSS dikatakan 'hanya untuk tampilan' karena CSS digunakan untuk mendesain halaman website yang dibangun dengan HTML. CSS sangat berguna dalam membuat tampilan halaman website menarik, mulai dari tampilan teks, gambar hingga latar belakang suatu halaman website.

30. 1.? Tuliskan pengertian website menurut pendapat anda. 2.) Jelaskan kegunaan dari website yang anda dapatkan. 3.) Tuliskan cara kerja dari hosting website sehingga pengguna internet dapatmengakses website tersebut.​


Jawaban:

Penjelasan:

1. Website adalah sebuah platform yang menyediakan informasi, konten, dan layanan yang dapat diakses melalui internet. Website dapat berupa halaman web statis atau dinamis yang dapat diakses melalui alamat URL.

2. Kegunaan website yang anda dapatkan tergantung pada jenis website yang anda buka. Misalnya, jika anda membuka website berita, anda dapat menemukan berita terbaru dan informasi lainnya. Jika anda membuka website toko online, anda dapat melihat produk yang dijual dan melakukan pembelian.

3. Hal yang paling utama dari suatu website menurut pendapat saya adalah desain dan navigasi. Desain website harus menarik dan mudah dipahami, serta navigasi harus jelas dan mudah diikuti. Ini akan membantu pengguna menemukan informasi yang mereka cari dengan cepat dan mudah.

4. Struktur layout dari website yang anda buka terdiri dari bagian atas, tengah, dan bawah. Bagian atas biasanya berisi logo, menu navigasi, dan tombol aksi. Bagian tengah berisi konten utama, seperti artikel, gambar, dan video. Bagian bawah berisi informasi tambahan, seperti alamat kontak, tautan ke media sosial, dan lainnya.

5. Cara kerja hosting website adalah dengan menyimpan semua file website di server. Server ini kemudian akan mengirimkan file tersebut kepada pengguna internet yang mengakses website tersebut. Ini memungkinkan pengguna untuk melihat dan mengakses website dari mana saja di dunia.
