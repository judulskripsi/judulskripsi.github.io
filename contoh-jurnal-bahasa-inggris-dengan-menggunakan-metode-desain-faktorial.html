13. Buatlah algoritma untuk menghitung nilai FAKTORIAL dari sebuah bilangan asli menggunakan metode looping! (GUNAKAN PSEUDOCODE)angka 5,6,7,8,9.


Berikut adalah pseudocode untuk menghitung nilai faktorial menggunakan looping:

FUNCTION faktorial(n: INTEGER): INTEGER

 result = 1

 FOR i = 2 TO n

   result = result * i

 ENDFOR

 RETURN result

ENDFUNCTION

Untuk menggunakan pseudocode ini, kamu hanya perlu memanggil fungsi faktorial dengan sebuah bilangan asli sebagai argumen. Misalnya:

faktorial(5)  // menghitung 5! (5 faktorial), hasilnya adalah 120

faktorial(6)  // menghitung 6! (6 faktorial), hasilnya adalah 720

faktorial(7)  // menghitung 7! (7 faktorial), hasilnya adalah 5040

faktorial(8)  // menghitung 8! (8 faktorial), hasilnya adalah 40320

faktorial(9)  // menghitung 9! (9 faktorial), hasilnya adalah 362880

Catatan: Faktorial dari sebuah bilangan n (n!) adalah hasil perkalian dari semua bilangan asli dari 1 sampai n. Misalnya, 5! (5 faktorial) adalah 1 x 2 x 3 x 4 x 5 = 120.

Apa yang dimaksud faktorial?

Penjelasan dengan langkah-langkah:

3! - 4⁵

=> ( 3 × 2 × 1 ) - ( 4 × 4 × 4 × 4 × 4 )

=> ( 6 × 1 ) - ( 16 × 4 × 4 × 4 )

=> 6 - ( 64 × 4 × 4 )

=> 6 - ( 256 × 4 )

=> 1.024

Apa yang dimaksud faktorial?

buatlah soal faktorial beserta contoh dan jawabannya!

Pembahasan:

_ _ _ _ _ _ _ _ _ _ _ _ _

Faktorial Adalah Bilangan Asli Dari " N " Ialah Perkalian Seluruh Bilangan Asli Yang Atau Sama Dengan N Faktorial Merupakan Perkalian Berulang Ulang Dengan Angka nya.

Suatu Bilangan Yang Ditandai Dengan Tanda Seru ( ! ) Bilangan Faktorial Dapat Dihitung Dari Angkanya Faktorial Dapat Diartikan Sebagai Perkalian Berulang Ulang Yang Sesuai Dengan Angka Tersebut.Faktorial di gunakan untuk menghitung dan menjumlahkan banyaknya susunan dari objek objek tersebut. Faktorial dari bilangan bulat positif dari n yang dilambangkan dengan n!

- Contoh Soal :

5! + 4! =

=> ( 5 × 4 × 3 × 2 × 1 ) + ( 4 × 3 × 2 × 1 )

=> ( 20 × 3 × 2 × 1 ) + ( 12 × 2 × 1 )

=> ( 60 × 2 × 1 ) + ( 24 × 1 )

=> ( 120 × 1 ) + 24

=> 120 + 24

=> 144
