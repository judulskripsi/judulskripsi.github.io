20. hasil query select sum (harga) from sepatu where harga < 600000 adalah​


Jumlah harga semua sepatu (dalam tabel) yang kurang dari 600000.

 

PembahasanQuery – SQL

Catatan:
Pada soal tidak ada spesifikasi data dan jenis database yang digunakan (MySQL, MariaDB, MS SQL Server, PostgreSQL, atau yang lainnya). Oleh karena itu, pembahasan ini berdasarkan SQL secara umum.

 

Diberikan query:

[tex]\large\text{$\begin{aligned}&\texttt{SELECT SUM(harga)}\\&\texttt{FROM sepatu}\\&\texttt{WHERE harga < 600000}\end{aligned}$}[/tex]

Perintah SQL SELECT ... FROM ... WHERE ... berfungsi untuk memilih atau menampilkan data sesuai field/kolom atau fungsi SQL pada argumen/parameter SELECT, yang diambil dari tabel pada argumen/parameter FROM, dan sesuai dengan kriteria pada argumen/parameter WHERE.

Pada perintah SQL di atas, terdapat fungsi SUM(<field>). Fungsi ini mengembalikan nilai “summation” atau penjumlahan dari seluruh data pada field parameternya. Jika ada kriteria pada bagian WHERE, maka hanya yang memenuhi kriteria saja yang dijumlahkan.

Sebagai contoh, berikut ini diberikan perintah SQL pembentukan dan pengisian data tabel “sepatu” secara sederhana berikut ini (hanya field “harga” saja).

[tex]\begin{aligned}&\texttt{CREATE TABLE sepatu (}\\&\quad\texttt{id INT AUTO\_INCREMENT,}\\&\quad\texttt{harga LONG,}\\&\texttt{);}\\&\texttt{INSERT INTO sepatu (harga)}\\&\texttt{VALUES}\\&\quad\texttt{(700000), (500000), (300000),}\\&\quad\texttt{(525000), (400000), (800000);}\end{aligned}[/tex]

Perintah query tersebut menciptakan sebuah tabel sepatu, dengan field/kolom "id" dan "harga", lalu diinisialisasi dengan 6 baris data.

Eksekusi query SELECT SUM(harga) FROM sepatu WHERE harga < 600000 akan menghasilkan:

[tex]\large\text{$\begin{aligned}&\texttt{SUM(harga)}\\&\texttt{1725000}\end{aligned}$}[/tex]

Jika kita jumlahkan sendiri semua harga pada tabel sepatu yang harganya kurang dari 600000:
500000+ 300000 + 525000 +400000
= 800000 + 925000
= 1725000

 

KESIMPULAN

Hasil query SELECT SUM(harga) FROM sepatu WHERE harga < 600000 adalah jumlah harga semua sepatu (dalam tabel) yang kurang dari 600000.
