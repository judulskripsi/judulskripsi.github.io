EXPLANATION

The text above tends to the personal recount. Personal recount text is a text which retells about the writer's past experience.

The generic structures of recount text :

1) Orientation is the very beginning paragraph which introduces the scene setting, what the event is, who involved in the story is, when the story happens, where the story takes a place and why it happens.

2) Sequences of event is the chronological order of the happened event.

3) Re-orientation is the last paragraph that contains the writer's conclusion/personal opinion.

The language features of recount text :

1) Using the past tense

2) Using the action verbs

3) Using temporal conjunction [after that, finally]
