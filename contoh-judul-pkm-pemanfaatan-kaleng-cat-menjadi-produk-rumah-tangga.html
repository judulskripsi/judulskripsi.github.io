14. untuk mengecat rumah pak anton memerlukan 2 kaleng cat dan 3 kuas seharga Rp 360.000,00. oleh karena ruangan yang harus di cat masih banyak, jadi ia membeli lagi 1 kaleng cat dan 1 kuas dengan harga Rp 170.000,00. harga 1 kaleng cat ialah...​


Jawaban:

Misal : x = cat; y = kuas

2x + 3y = 360.000...(1)

x + y = 170.000...(2)

•> 2x + 3y = 360.000

x + y = 170.000

x = 170.000 - y...(3)

• Substitusi x ke persamaan 1

=> 2x + 3y = 360.000

=> 2(170.000 - y) + 3y = 360.000

=> 340.000 - 2y + 3y = 360.000

=> 340.000 - y = 360.000

=> y = 360.000 - 340.000

=> y = 20.000

• Substitusi y ke pers. 3

=> x = 170.000 - y

=> x = 170.000 - 20.000

=> x = 150.000

Maka harga satu kaleng cat adalah Rp. 150.000,-

Campuran mana yang menghasilkan warna hijau paling pekat?* Mencampurkan 2 kaleng cat hijau dengan 3 kaleng cat putih Mencampurkan 5 kaleng cat hijau dengan 9 kaleng cat putih Mencampurkan 1 kaleng cat hijau dengan 2 kaleng cat putih Mencampurkan 3 kaleng cat hijau dengan 5 kaleng cat putih ​


Jawaban:

cara menghitung pecahan menjadi desimal ada pada lampiran.

Penjelasan dengan langkah-langkah:

PEMBIASAN WARNA

Untuk mengetahui tingkat kepekatan suatu warna kita dapat membandingkan antara warna yang dicampur dengan warna netral

Mencampurkan 2 kaleng cat hijau dengan 3 kaleng cat putih = ⅔ = 0,67Mencampurkan 5 kaleng cat hijau dengan 9 kaleng cat putih = 5/9 = 0,56Mencampurkan 1 kaleng cat hijau dengan 2 kaleng cat putih = ½ = 0,50Mencampurkan 3 kaleng cat hijau dengan 5 kaleng cat putih = 3/5 = 0,6

Dari hasil penghitungan, didapat nilai desimal terbesar adalah 0,67

Jadi untuk menghasilkan Campuran warna hijau paling pekat, dengan Mencampurkan 2 kaleng cat hijau dengan 3 kaleng cat putih ✔
