Nilai tersebut diperoleh dari perhitungan sistem persamaan linier tiga variabel dengan metode subtitusi. SImak pembahasan berikut.

Pembahasan

Diketahui:

Sebuah tempat parkir menyediakan  lahan parkir untuk motor, mobil, dan

minibus

Ruang parkir untuk minibus lima  kali lebih banyak dari motor

Ruang parkir yang tersedia  untuk semua jenis kendaraan adalah 600

Jawab:

Misalkan:

Ruang parkir untuk motor = x

Ruang parkir untuk mobil = y

Ruang parkir untuk minibus = z

Maka diperoleh persamaan sebagai berikut:

z = 5x ........................1)

y = 3(x + z) ...............2)

x + y + z = 600 ........3)

subtitusikan persamaan 1) ke persamaan 2)

y = 3(x + z)

y = 3(x + 5x)

y = 3(6x)

y = 18x ....................4)

subtitusikan persamaan 1) dan 4) ke persamaan 3)

x + y + z = 600

x + 18x + 5x = 600

24x = 600

x = 600/24

x = 25

Subtitusikan nilai x ke dalam persamaan 4)

y = 18x

y = 18 × 25

y = 450

Subtitusikan nilai x kedalam persamaan 1)

z = 5x

z = 5 × 25

z = 125

Sehingga diperoleh penyelesaian sebagai berikut:

Banyak ruang parkir untuk motor = 25

Banyak ruang parkir untuk mobil = 450

Banyak ruang parkir untuk minibus = 125

34. Di tempat parkir sebuah rumah sakit terdapat 32 kendaraan yg terdiri dari mobil dan sepeda motor. Banyaknya roda seluruhnya 88. Jika tarif parkir untuk mobil 5.000 dan sepeda motor 3.000, maka pendapatan uang parkir saat itu adalah..


Misalkan banyaknya motor adalah x dan banyaknya mobil adalah y, maka:

x + y = 32 ... (1)
2x + 4y = 88 ... (2)

Eliminasi x, didapat:
2x + 4y = 88
2x + 2y = 64
_________ _
2y = 24
y = 12

Kemudian kita substitusikan y ke persamaan 1, maka:
x + y = 32
x + 12 = 32
x = 32 - 12
x = 20

Maka pendapatan uang parkir saat itu adalah:
3000x + 5000y
= 3000(20) + 5000(12)
= 60.000 + 60.000
= 120.000
