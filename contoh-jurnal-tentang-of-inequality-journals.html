4. If a > 0, prove the inequality, a+a^-1 ≥ 2, a>0.


Pembahasan

Properties of positive real numbers

( Sifat-sifat bilangan real positif )

[tex]a, b > 0 \: \: \: maka \\ \\ {a}^{2} > 0 \: \: dan \: \: {b}^{2} > 0 \\ \\ {(a - b)}^{2} \geqslant 0 \\ \\ \\ {(a - b)}^{2} \geqslant 0 \\ \\ {a}^{2} + {b}^{2} - 2ab \geqslant 0 \\ \\ {a}^{2} + {b}^{2} \geqslant 2ab \\ \\ \\ untuk \: \: a > 0 \: \: dan \: \: b = 1 \\ \\ {a}^{2} + {b}^{2} \geqslant 2ab \\ \\ {a}^{2} + 1 \geqslant 2a \\ \\ \large{ \frac{ {a}^{2} + 1}{a} \geqslant \frac{2a}{a} } \\ \\ \frac{ {a}^{2} + 1}{a} \geqslant 2 \\ \\ \boxed{ \large{a + \frac{1}{a} \geqslant 2} } \\ \\ [/tex]

( Proved )

20. 10 A Mathematics examination consists of Papers I, II and III. The full score of each paper is 100. In order to pass the examination, the average score of all three papers must be at least 60. A student scores 57 marks in Paper I. (a) Ifx is the average score of Paper II and Paper III, write down an inequality in x such that the student passes the examination. (b) Hence solve the inequality to find the minimum value of x.


Penjelasan dengan langkah-langkah:

(a) To pass the examination, the average score of all three papers must be at least 60. Let's assume the average score of Paper II and Paper III as x.

The average score of the three papers can be calculated as (57 + x) / 3.

To pass the examination, this average score must be greater than or equal to 60. Therefore, we can write the inequality:

(57 + x) / 3 ≥ 60

(b) To solve the inequality and find the minimum value of x, we can follow these steps:

Multiply both sides of the inequality by 3 to eliminate the denominator:

57 + x ≥ 180

Subtract 57 from both sides of the inequality:

x ≥ 180 - 57

x ≥ 123

Therefore, the minimum value of x is 123. This means the student needs to score at least 123 on average in Paper II and Paper III to pass the examination.

34. If 0 < a < 1, so the solution set of inequality a is...​


If 0 < a < 1, the solution set of inequality a is all real numbers that are greater than 0 and less than 1.

In other words, the solution set of the inequality a is the interval (0,1), which includes all real numbers x such that 0 < x < 1.

For example, if a = 0.5, then the solution set of the inequality a is the interval (0,1), which includes all real numbers x such that 0 < x < 1. This interval includes all numbers between 0 and 1, including 0.5.

If a is any other number between 0 and 1, the solution set of the inequality a will also be the interval (0,1).
