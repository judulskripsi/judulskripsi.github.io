15. Tembung-tembung ing ngisor iki wuwuhana seselan (sisipanEsem + (-um)Elar + (-um)Laku + (-um)Gagah + (-um)Seleh + (-um)Selang+ (-um)Bagus + (-um)Pinter + (-um)panggang + (-um)0. gedhe + (-um)​


WangsulanEsem + um → mesem.Elar + um → melar.Laku + um → lumaku.Gagah + um → gumagah.Seleh + um → sumeleh.Selang + um → sumelang.Bagus + um → bumagus → gumagus.Pinter + um → puminter → kuminter.Panggang + um → pumanggang → kumanggang.Gedhe + um → gumedhe.

Penjelasan

Tembung Andhahan

→ Tembung andhahan iku tembung kang wis owah saka asale utawa linggane. Owahe bisa amarga diwenehi ater-ater, seselan, panambang, utawa dirangkep. Tembung lingga kang wis diowahi iku bakal malih dadi tembung andhahan.

☞ Seselan yaiku wuwuhan sing manggon ana ing tengahe tembung. (in, um, er, el)

✧ Tuladha :

Kumentus → um + kentus.Tumumpang → um + tumpang.Lumebu → um + lebu.Gumantung → um + gantung.Sumeleh → um + seleh.Sinapa → in + sapa.Sinamber → in + samber.Ginanjar → in + ganjar.Cinarita → in + carita.Binoyong → in + boyong.

> El + kepyur » kelepyur.

> Um + pinter » puminter → kuminter.
